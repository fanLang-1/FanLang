# 🚀 5 分钟上手

## 第 1 分钟：装好环境

### Windows
```bash
# 1. 下载 Ollama
https://ollama.com/download/windows

# 2. 安装后，打开终端（Win+R → cmd）
ollama pull qwen2.5:1.5b

# 3. 装 Python
winget install Python.Python.3.12

# 4. 启动工具
pip install streamlit
streamlit run apps/scaffold.py
```

### Mac
```bash
brew install ollama
ollama pull qwen2.5:1.5b
pip3 install streamlit
streamlit run apps/scaffold.py
```

## 第 2 分钟：认识 7 个字

打开界面后，你会看到 12 个大按钮。记住 7 个就够了：

**写 改 翻 总 问 画 想**

## 第 3 分钟：用一次

1. 点【写】
2. 输入："帮我写一段自我介绍，我是做服装生意的"
3. 等 3 秒，看结果

## 第 4 分钟：换着用

试试这些组合：

- 【改】"这段话太正式了，改成口语"
- 【总】把一篇长文章贴进去
- 【问】"为什么天空是蓝色的？"
- 【想】"给我 5 个促销方案"

## 第 5 分钟：你会了

你已经会用 AI 了。就这么简单。

# 📖 中文 AI 符号 · 完整表

## 第一梯队 · 7 个核心（日常够用）

| 符号 | 名称 | 用法 |
|:----:|:----|:-----|
| ✍️ **写** | 写作 | 写文章、写报告、写文案、写代码 |
| ✏️ **改** | 改写 | 改文字、改风格、改语气、改长度 |
| 🔄 **翻** | 翻译 | 中译英、英译中，或其他语言 |
| 📋 **总** | 总结 | 提炼重点、压缩长文为摘要 |
| ❓ **问** | 问答 | 不懂就问，知无不答 |
| 🎨 **画** | 绘画 | 描述画面，生成绘图提示词 |
| 💡 **想** | 创意 | 头脑风暴、出点子、给灵感 |

## 第二梯队 · 5 个进阶

| 符号 | 名称 | 用法 |
|:----:|:----|:-----|
| 🔍 **解** | 解释 | 复杂概念用人话说清楚 |
| 📚 **教** | 教学 | 从零开始一步步教你 |
| 📊 **比** | 比较 | 两个东西全方位对比 |
| 🔎 **查** | 检索 | 在你的资料里搜答案 |
| 🗂️ **理** | 整理 | 分类、排序、结构化信息 |

## 第三梯队 · 8 个专项

| 符号 | 名称 | 用法 |
|:----:|:----|:-----|
| 💻 **编** | 编程 | 写代码、改 bug、解释代码 |
| 🔄 **转** | 转换 | 格式转换、风格转换 |
| 🧮 **算** | 计算 | 数学计算、数据分析 |
| 📝 **记** | 记忆 | 记住重要信息 |
| ⏰ **定** | 定时 | 规划时间、安排任务 |
| 🎤 **说** | 语音 | 文字转口语、朗读 |
| 🧪 **试** | 尝试 | 试错、实验、A/B测试 |
| ⭐ **评** | 评价 | 打分、评判、给反馈 |

## 组合示例

```
【写】+【改】 = 先写再改
【查】+【总】 = 搜索并总结（这就是 RAG）
【问】+【解】 = 提问并用人话解释
【翻】+【改】 = 翻译后润色
【想】+【写】 = 先出点子再写成文章
【编】+【试】 = 写代码再测试
【比】+【评】 = 对比并评价优劣
```

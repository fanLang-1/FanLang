# 为凡语贡献代码

欢迎！凡语的使命是**用中文单字重新定义 AI 交互界面**。

## 如何贡献

### 提 Bug

1. 确认问题在最新版本可复现
2. 提供最小复现代码
3. 标注操作系统和 Python 版本

### 提 PR

1. Fork 仓库
2. 创建 feature 分支
3. 写代码（加测试）
4. 确保 `python -m pytest` 通过
5. 提交 PR

### 添加新符号

如果你认为有新的汉字应该加入凡语符号体系：

1. 先在 Discussion 发起讨论
2. 说明使用场景和分类理由
3. 通过后修改 `fanlang/symbols.py`

### 代码风格

- 完整类型提示
- 中文 docstring
- 行宽 100 字符

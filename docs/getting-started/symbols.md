# 符号体系

凡语的核心是 **一套中文符号系统**——用单字和双词表达 AI 能力。

## 25 个单字符号

分 4 个梯队，5 个分类：

| 分类 | 符号 |
|:--|:--|
| 内容创作 | ✍️ 写 🔄 改 ✏️ 编 🎨 画 📝 记 |
| 信息处理 | 🌐 翻 📝 总 🔍 查 🗂️ 理 🧹 清 |
| 分析推理 | ❓ 问 💡 想 ⚖️ 比 🔬 研 🧮 算 |
| 编程开发 | 💻 码 🐛 修 🚀 发 🔧 配 📋 测 |
| 辅助工具 | ⏰ 时 📅 排 📎 附 🔗 关 🔔 提 |

查看完整列表：`get_all_symbols()`

## 核心 7 符号

日常最常用的 7 个符号：

```
✍️ 写  🔄 改  🌐 翻  📝 总  ❓ 问  🔍 查  ⚖️ 比
```

## 36 个双词符号

按 6 类组织，用于更精细的能力表达：

- 写作类：写文章、改风格、翻文件...
- 分析类：问因果、比优劣、解难题...
- 编程类：码功能、修错误、配环境...

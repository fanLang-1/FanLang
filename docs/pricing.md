## 企业版详情

### 适合谁

- 正在开发 AI 产品的创业团队（3-20 人）
- 需要将符号栏嵌入自己产品的技术团队
- 需要私有化部署的 B 端客户

### 包含服务

1. **快速集成支持** — 一对一远程协助，1 个工作日内完成接入
2. **私有部署文档** — Docker / K8s / 内网环境适配指南
3. **符号体系定制** — 可增删改符号，匹配产品自身能力
4. **专属更新通道** — 优先获取内部版本和 hotfix
5. **发票 + 合同** — 企业正规流程

### 典型场景

```
SaaS 产品 "AI 写作助手" 接入凡语：

安装 → 定制符号（写/改/翻/总→改成自己产品名称）
      → render_symbol_bar() 嵌入页面
      → OODA 自动追问 → 用户直接上手
      → 整体集成时间：< 1 天
```

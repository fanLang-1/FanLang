# 关于凡语

## 为什么叫「凡语」？

- **凡** — 平凡、平易，降低 AI 的使用门槛
- **语** — 语言、符号，用中文表达 AI 意图

合起来：**用最平凡的语言，操作最不平凡的技术。**

## 项目哲学

1. **交互层独立** — AI 与 UI 之间的中间层，不绑定任何模型
2. **符号驱动** — 用字符而非 prompt 控制 AI，更直观、更可复现
3. **组织即架构** — 多智能体不是对话链，是组织设计

## 技术栈

- Python >= 3.10
- Streamlit / LangChain（可选插件）
- Ollama / 任何 OpenAI 兼容 API
- ChromaDB（可选 RAG）
